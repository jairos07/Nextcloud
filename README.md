# Nextcloud Server Setup

Este repositorio documenta el proceso de **instalación, configuración y puesta en marcha de un servidor Nextcloud**, una plataforma de almacenamiento en la nube autoalojada, segura y de código abierto.

El objetivo del proyecto es servir como **guía práctica y reproducible** para desplegar Nextcloud en un servidor propio.

---

## ¿Qué es Nextcloud?

Nextcloud es una solución de **cloud privada** que permite:

- Almacenamiento y sincronización de archivos
- Calendario y contactos
- Compartición segura de archivos
- Colaboración en tiempo real
- Control total de tus datos

---

## Tecnologías utilizadas

- **Sistema operativo:** Linux (Ubuntu / Debian)
- **Servidor web:** Apache / Nginx
- **Base de datos:** MariaDB / MySQL
- **Lenguaje:** PHP
- **Nextcloud:** Última versión estable
- **xampp:**Despliegue de servidor


---

## Requisitos previos

- Servidor o máquina virtual
- Acceso root o sudo
- Conexión a internet
- Dominio o IP fija (opcional pero recomendado)
